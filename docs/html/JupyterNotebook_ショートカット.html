Jupyter Notebook ショートカット集

Jupyter Notebookは、キーボードショートカットに慣れれば非常に作業が高速化できますが、それらを覚えるのが少し大変です。

OSごとにキーボードショートカットが異なるため、それぞれのOSでのキーボードショートカットをまとめたページを紹介します。

Jupyter Notebook上でもコマンドモード（編集モードではない）でhと打てば、お使いの環境でのキーボードショートカット一覧が出てきます。

Jupyter Notebookのキーボードショートカット（Windows）

0 1
0 F find and replace
1 Ctrl-Shift-F open the command palette
2 Ctrl-Shift-P open the command palette
3 Enter enter edit mode
4 P open the command palette
5 Shift-Enter run cell, select below
6 Ctrl-Enter run selected cells
7 Alt-Enter run cell, insert below
8 Y to code
9 M to markdown
10 R to raw
11 1 to heading 1
12 2 to heading 2
13 3 to heading 3
14 4 to heading 4
15 5 to heading 5
16 6 to heading 6
17 K select cell above
18 Up select cell above
19 Down select cell below
20 J select cell below
21 Shift-K extend selected cells above
22 Shift-Up extend selected cells above
23 Shift-Down extend selected cells below
24 Shift-J extend selected cells below
25 A insert cell above
26 B insert cell below
27 X cut selected cells
28 C copy selected cells
29 Shift-V paste cells above
30 V paste cells below
31 Z undo cell deletion
32 D,D delete selected cells
33 Shift-M merge selected cells, or current cell with cel...
34 Ctrl-S Save and Checkpoint
35 S Save and Checkpoint
36 L toggle line numbers
37 O toggle output of selected cells
38 Shift-O toggle output scrolling of selected cells
39 H show keyboard shortcuts
40 I,I interrupt kernel
41 0,0 restart the kernel (with dialog)
42 Esc close the pager
43 Q close the pager
44 Shift-L toggles line numbers in all cells, and persist...
45 Shift-Space scroll notebook up
46 Space scroll notebook down

Linux

LinuxでもWindowsと同じキーボードショートカットで行けると思います。

便利なショートカット

  • コマンドモードでD,D（2回Dを打つ）とセルが消えます。間違って消したときにはZでもとに戻せます。
  • コードを選択し、Windowsなら「Ctrl + /」、macOSなら「Cmd + /」でコメントアウト（と解除）ができます。
