科学技術計算のためにPythonを導入したいと考えている人のためのページです。

「とりあえずPythonを使ってみたい」という方は、Anacondaをインストールして、SpyderやIPythonなどの実行環境を手に入れてください。

実際にコードを実行して確かめながら、Pythonの基本的な文法を抑え（Pythonの基礎1,2）、数値計算用ライブラリのNumPyの記事（NumPyの基礎）と、プロット可視化ライブラリのMatplotlibの記事（Matplotlib）を読めば、それなりのことが出来るようになります。
