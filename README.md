# Kyoto-GeoPython

## convert ipynb to html

```
$ make convert
```

## run local server

```
$ make start
```

Open `http://localhost:8080` in your browser.

# setup

Install Poetry.

https://python-poetry.org/docs/

Or `brew install poetry` for Mac OS.


```
asdf install
poetry install
```

## Tips

`.venv` in repository root directory.

```
poetry config virtualenvs.in-project true
```
